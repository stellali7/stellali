About me

I am a 4th year undergraduate student at Johns Hopkins University. I major in Computer Science, Cognitive Science (linguistics focus), and Applied Mathematics (statistics focus). I'm intereted in using computational methods to model and potentially discover cognitive processes.

My research interests: Natural Language Processing, Computational Sociolinguistics, Multilingual Low-Resource Machine Translation, Human-Centered NLP, Ethical NLP

I am in the process of applying to post-bac and PhD programs. Please contact me at sli136@jhu.edu if you are interested in my work!

    • Current Projects

    • Multilingual L2 Translation

      We develop a multilingual translation system from "broken" source language (potentially from an L2 learner) to target languages by fine-tuning multilingual PLMs.

    • Code-Switching

      We introduce a flexible code-mixing generation algorithm to synthesize language-agnostic code-mixing sentences as a data augmentation for sentiment analysis tasks.

    • Sign Language

      Using distillation models to translate speech to animated ASL sign language. End-to-end signed language translation for different types of sign languages.

    • Quantum Language Modeling

      Integrating VQCs into a lite Transformer architecture for fast and secure language model pre-training. We demonstrate the ad hoc portability of the quantum language models.

    • Self-Supervised Models for Multilingual ASR

      We investigate the cross-lingual generalizability of self-supervised models trained on English as a feature extractor for ASR tasks from a linguistics perspective. We use DeepCCA to perform semantic-phonology separation for language-agnostic acoustic modeling.

    Projects

    • Language Processing

      May 22 - Current

      Multilingual Generalizability of Self-Supervised Models

      We use self-supervised models trained on English speech data as a feature extractor for ASR tasks in other languages and perform semantic-phonology separation on the extracted features.

    • Language Processing

      May 22 - Current

      Language Agnostic Code-Mixing Data Augmentation

      We establish a language-agnostic synthetic code-mixing algorithm that generates code-mixing datasets in a zero-cost fashion.

    • Language Processing

      Mar. 22 - Oct. 22

      Quantum Language Model

      We propose a quantum language model that can easily transmit learned information to downstream classical models, demonstrating ad hoc model portability.

    • Speech Processing

      Feb. 22 - Oct. 22

      Self-Supervised Learning for Lyric Recognition

      We design an end-to-end lyrics recognition model using self-supervised models trained on spoken speech and out-performed previous SOTA.

    • Machine Learning

      Sep. 20 - Current

      Delineo Covid-19 Modeling

      We design an expansion-based clustering algorithm for community detection using covid-19 mobility data to optimize disease simulations model.

    • Genetic Programming

      Jul. 22 - Current

      Cryptographic Code Optimization

      This project extends on the Shackleton project and optimizes cryptographic code generation with genetic improvement. We aim to automatically reduce the computation time for encryption algorithms.

    • Signal Processing

      Mar. 22 - Oct. 22

      Fetal Electrocardiogram Extraction

      This project uses a convex combination of adaptive filters to detect the fetal heart rate from the ECG signals from the mother's abdominal and thorax. Our algorithm out-performs both single adaptive filters and convex combination of RLS filters

    • Genetic Programming

      May 21 - Jul. 22

      Genetic Improvement - Patch Algorithm

      We present the novel use of genetic improvement to find problem-specific optimized LLVM Pass sequences. This demonstrates the flexibility of the Shackleton Framework in being applied to a variety of use cases.

    • Genetic Programming

      May 21 - Jul. 22

      Genetic Programming - LLVM Optimization
